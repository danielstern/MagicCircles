Styling

You can style a Magic Circle as a whole or style each element individually.

Default Styles

        var magic = new MagicCircle("#stylingExample");
        magic.styles.colors.ring = "blue";
        magic.styles.colors.text = "red";
        magic.styles.colors.smallRing = "green";

        magic.cast()
          .circleRing(12,12)
          .ring(10)
          .text(16,"Red Text")

Styling Individual Elements

        var magic = new MagicCircle("#stylingExample");
        magic.cast()
          .circleRing(12,12)
            .color('red')
          .cirlceRing(14,14)
            .color('blue')
            .fill('yellow')
