Layering

You can layer several elements on top of each other for cool effects.

Layering Text over Rings

You can layer text or other elements over one another like in the example below.

  var magic = new MagicCircle("#layeringExample");
magic.cast()
  .space(50)
  .ring(25)
  .color("magenta")
  .backspace(20)
  .text(20, "DISCO PARTY")
  .color("white");

Inverting Elements

By layering elements the same color as the background over others, you can create interesting patterns.

    var magic = new MagicCircle("#layeringExample");
  magic2.cast()
    .space(50)
    .ring(25)
    .color('blue')
    .ring(25)
    .color('white')
    .backspace(20)
    .text(15, "SAILORS ENTHUSIAST MEETUP GROUP", 1, true)
    .color("blue")
    .ring(40)
    .color('blue') // draw a blue ring
    .backspace(42)
    .circleRing(8, 30, 2) // draw white circles on top of it
    .fill("white")
    .color("white")
