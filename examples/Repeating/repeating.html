Repeating Pattern

Magic Circles is ideal for randomly generating dynamic images. They are like snowflakes! 

         var drawRandomCircle = function(count) {

      magic.cast()
        .space(50)
        .ring()
      for (var i = 0; i < count; i++) {

        magic.cast()
          .text("autofit", lol.physics()) 
            .color(lol.randomColor())
          .ring()
